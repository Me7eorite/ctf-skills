# Shared Generation Strategy

Use the same search and synthesis rules for both research and design:

- Keep search breadth wide; do not narrow the topic just to satisfy a retry.
- Prefer authoritative or directly technical sources over low-signal summaries.
- Reuse already confirmed stable findings before opening a new exploration branch.
- When a run hits budget, finalize from consulted evidence instead of starting over.
- Preserve distinct sub-techniques, but do not require one unique finding per target slot.
- Treat repeated deterministic outputs as reusable evidence, not fresh work to re-discover.
