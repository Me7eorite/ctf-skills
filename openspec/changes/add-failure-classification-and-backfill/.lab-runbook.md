## 9.3 单条 dry-run（lab）

```bash
# 1. 在 lab DB 找一条 failed 且 hermes_log_path 不空的 fixture run
psql "$DATABASE_URL" -c "
  SELECT id, last_error, hermes_log_path
  FROM research_runs
  WHERE status = 'failed' AND hermes_log_path IS NOT NULL
  ORDER BY finished_at DESC
  LIMIT 5;"

# 2. 选一个 run_id，跑 dry-run
uv run challenge-factory research backfill --run-id <UUID> --dry-run

# 3. 核对零变化
#    应该看到 [backfill] would_insert_sources=N would_insert_findings=M log_sha256=...
#    DB 上 sources_count / findings_count 应未变动
psql "$DATABASE_URL" -c "
  SELECT COUNT(*) FROM research_sources WHERE research_run_id = '<UUID>';"
psql "$DATABASE_URL" -c "
  SELECT COUNT(*) FROM research_findings WHERE research_run_id = '<UUID>';"
#    文件系统应未变动
ls -la work/research/sources/<UUID>/ 2>/dev/null || echo "no final dir (expected)"
ls -la work/research/sources_staging/<UUID>/ 2>/dev/null || echo "no staging dir (expected)"
```
